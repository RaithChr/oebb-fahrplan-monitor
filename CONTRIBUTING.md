# Contributing to ÖBB Fahrplan Monitor

Vielen Dank für dein Interesse am Projekt! 🎉

## 🤝 Wie kann ich beitragen?

### Bug Reports
- Öffne ein [Issue](https://github.com/RaithChr/oebb-fahrplan-monitor/issues)
- Beschreibe das Problem detailliert
- Füge Screenshots hinzu (falls relevant)
- Gib deine Browser-Version an

### Feature Requests
- Öffne ein [Issue](https://github.com/RaithChr/oebb-fahrplan-monitor/issues)
- Beschreibe das Feature und den Use Case
- Erkläre warum es sinnvoll wäre

### Pull Requests
1. **Fork** das Repository
2. **Clone** dein Fork lokal
3. Erstelle einen **Feature Branch**: `git checkout -b feature/mein-feature`
4. **Committe** deine Änderungen: `git commit -m 'Add: Mein Feature'`
5. **Pushe** zum Branch: `git push origin feature/mein-feature`
6. Öffne einen **Pull Request**

## 📝 Code Style

### PHP
- PSR-12 Coding Standard
- UTF-8 Encoding
- Input-Validierung mit `trim()` und `filter_var()`

### JavaScript
- ES6+ Syntax
- Keine externen Dependencies (Vanilla JS)
- Saubere Namensgebung (camelCase für Variablen)

### CSS
- BEM-ähnliche Namenskonvention
- Mobile-First Approach
- Kommentare für komplexe Selektoren

## 🧪 Testing

Bitte teste deine Änderungen in:
- ✅ Chrome (Desktop & Mobile)
- ✅ Safari (macOS & iOS)
- ✅ Firefox
- ✅ Edge

## 🎯 Commit Messages

Verwende aussagekräftige Commit Messages:

```
Add: Neue Feature XYZ
Fix: Bug in ABC behoben
Update: Verbesserung von DEF
Docs: README aktualisiert
Style: CSS Formatierung
Refactor: Code-Optimierung
```

## ⚠️ Wichtige Hinweise

- Keine API-Keys oder Credentials committen
- Keine großen Binärdateien (max 1 MB)
- Screenshots in `/docs/` Ordner
- UTF-8 Encoding für alle Dateien
